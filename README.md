К.Р.И.П. – Комитет Расследования Иных Пространств

Описание:
  Проект представляет собой официальную базу данных К.Р.И.П. – Комитета Расследования Иных Пространств. Содержит классификацию и протоколы содержания аномальных объектов, известных как "крипы". Все данные получены в ходе наблюдений и экспериментов, проводимых сотрудниками Комитета.

Особенности базы данных:

  - Полная регистрация крипов с уникальными идентификаторами (ID).
  - Система классификации по уровню угрозы: NULL, FLUX, RUPTURE, DISTORT.
  - Протоколы содержания для безопасного взаимодействия с аномалиями.
  - Возможность поиска по ID и названию объекта.
  - Контроль доступа к объектам с ограничением видимости особо опасных крипов.

Классы аномалий:

  Класс       Описание                                                                     
  ________________________________________________________________________________________
  NULL        Объекты безопасного содержания, не представляющие угрозы.                    
  FLUX        Аномалии, которые могут нарушать восприятие или порядок, требуют наблюдения. 
  RUPTURE     Опасные объекты, способные причинить физический или психический вред.        
  DISTORT     Аномалии, искажающие пространство или восприятие, не атакуют напрямую.       

Протокол использования:
  - Все наблюдения объектов ведутся через камеры и датчики, прямой контакт ограничен.
  - Доступ к информации регулируется уровнями допуска.
  - Любые проявления аномалий фиксируются и анализируются сотрудниками К.Р.И.П.

Цель проекта:
  - Создание точного и достоверного архива аномальных объектов для обеспечения безопасности персонала и изучения искажений пространства и восприятия.

Предупреждение:
  - Некоторые объекты крайне опасны. Любое нарушение протоколов содержания может привести к непредсказуемым последствиям.
